# STORM PS4 PKG SENDER
- Автоматическое подключение ПК к PS4 по Wi-Fi / Ethernet по портам 12800 (RPI FlatZ) или 12801 (RPI OOP). Один из RPI должен быть запущен на PS4!
- Поддержка ручного ввода IP и выбора из нескольких PS4.
- Удаленная установка .pkg-файлов на PS4.
- Drag-and-drop папок или отдельных файлов, а также через кнопку "Добавить" (на выбор: Файлы (.pkg) или Папка).
- Закрепление папок с сохранением прогресса и статуса всех файлов при закрытии. Возможность скрыть закрепленные папки.
- Русский и английский интерфейс.
- Крупный шрифт - увеличивает шрифт без изменения размеров интерфейса.
- 30 тем оформления.
- Возможность установки и удаления своего фона.
- Общий индикатор установки.
- Глобальные кнопки: Свернуть все - сворачивает все папки; Развернуть все - разворачивает все папки; Пауза все - ставит все файлы на паузу; Отменить все - отменяет все установки; Установить все - устанавливает все файлы, ранее не установленные.
- Бэкап по FTP (порт 2121) с возможностью отключения, указания папки для бэкапов и интервала проверки, а также ручным запуском. При включенном авто-бэкапе - создается каждый раз при открытии программы.
- Восстановление бэкапа - возможность выбрать нужную папку с бэкапом (по умолчанию открывается общая папка, указанная для бэкапов) и выбрать нужные файлы для восстановления (по Title ID).
- При добавлении файлов / папок, автоматически считывается Title ID, Версия и Размер файлов.
- Отображение скорости установки файлов на PS4.
- Динамическое отображение прогресса установки.
- Статусы установки.
- Проверка ранее установленных игр (в таком случае переводит в статус "Ранее установлено"). Внимание! Обновления и дополнения все равно могут начать устанавливаться, т.к. это особенность самой PS4!
